# Data Representation and Operations
## Base (Radix)
- **r-ary system**
 $$K_{n} K_{n-1} K_{n-2} \dots K_{0} K_{-1} \dots K_{-m}$$    
数值表示 $$K_{n} r^{n} + K_{n-1} r^{n-1} + \dots + K_{0} r^{0} + K_{-1} r^{-1} + \dots + K_{-m} r^{-m} = \sum_{i=n}^{-m} K_{i} r^{i}$$
- 二进制  
$$01 \space r=2$$
- Octal
$$01234567 \space r=8=2^3$$
- 十六进制  
$$0123456789ABCDEF \space r=16=2^4$$
## Base Conversion
4-bit binary digits correspond to 1-digit hexadecimal number
3-bit binary digits correspond to 1-digit octal number
